Login to UP IRIS

1.       Go to the Admin button to login.

2.       Enter the required fields, Username and Password.

 

3.       Click the Admin login button and proceed.

 4.       You have successfully logged in if you can see your modules or functions.

Editing Login Details

1.       Click on the option Edit Login Details to go to the page where you can update your login data.

2.       Update your login data by editing the fields shown on the page.

3.       When you are done changing your details, click the Save button to save.

4.       To cancel updating, click the Cancel button.

Editing User Profile

1.       Click on the option Edit User Profile to go to the page where you can update your login data.

2.       Update your user profile data by editing the fields shown on the page.

3.       When you are done changing your details, click the Save button to save.

4.       To cancel updating, click the Cancel button.

  

STUDENTS ACCOUNTABILITY MODULE

                The student’s accountability module for the Library is concerned with accountabilities that are related to unreturned books, damaged books, and fines. This module allows the user to manage the accountabilities of students that are of the types stated earlier, functions include add, edit, clear, and view.

 

Add Accountability

1.       Click the Students Accountability Module, then click on  Add Accountability.

 

2.       Enter the student number or last name of student on the input box then click the Submit button.

 

 

3.       If student number is entered, fill up the fields, which ask for the details of the accountability you are adding then press the Save button.

4.       If last name is entered, possible entries of students with the last name entered will appear. Choose the correct student and press Use.

5.       Refer to no.3: Fill up the fields, which ask for the details of the accountability you are adding then click the Save button.

 

Edit Accountability

1.       Click the Students Accountability Module.

2.       Enter the last name of the student on the search bar and click the Search button.

      3.       Choose the correct student then click Edit.

 

4.       Update the details you need to then click the Save button.

 

Clear Accountability

1.       Click the Students Accountability Module.

2.       Enter the last name of the student on the search bar and click the Search button.

 

3.       Choose the correct student then click Clear.

 

4.       Input the details  on the fields required then click the Clear button.
