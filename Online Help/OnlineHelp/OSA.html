Add Accountability:

  • To add an accountability, Go to: Osa staff functions
  • Click: Student Accountability Module
  • Go to Add Student Accountability
  • Fill in all the required fields
  • Click "Add Accountability" button

    View Accountability:

  • To view student accountability, Go to: Osa staff functions
  • Click: Student Accountability Module
  • Go to View Student Accountability
  • Enter Student Number
  • Click "View Accountability" button

    View Scholarship List

  • To view a scholarship, Go to: Osa staff functions
  • Click: Scholarship Management Module
  • Go to View Scholarship List

    Adding Approved Scholarship

  • To add a scholarship, Go to: Osa staff functions
  • Click: Scholarship Management Module
  • Go to Add Approved Scholarship

    Removing Scholarship

  • To remove a scholarship, Go to: Osa staff functions
  • Click: Scholarship Management Module
  • Go to Remove Scholarship

    Modifyng Scholarship Information

  • To modify a scholarship, Go to: Osa staff functions
  • Click: Scholarship Management Module
  • Go to Edit Scholarship Information

    Adding Student's Scholarship

  • To add a scholarship to a student, Go to : Osa Staff functions
  • Click: Student Scholarship Module
  • Go to Add Student Scholarship
  • Enter the Student Number and Scholarship
  • Click: Add Scholarship button

    Modifyng Student's Scholarship

  • To modify a student's scholarship, Go to : Osa Staff functions
  • Click: Student Scholarship Module
  • Go to Edit Student Scholarship
  • Enter the Student Number in the search box
  • Click: Search Student button
  • Click: Edit Scholarship button
  • Enter all appropriate data
  • Click: Edit Student Scholarship button

    Removing Student's Scholarship

  • To remove a student's scholarship, Go to : Osa Staff functions
  • Click: Student Scholarship Module
  • Go to Remove Student Scholarship
  • Enter the Student Number in the search box
  • Click: Remove Scholarship button
  • Click the YES button to proceed

    Adding STFAP Bracket

  • To add an STFAP Bracket, Go to : Osa Staff functions
  • Click: STFAP Bracket Management Module
  • Go to Add STFAP Bracket
  • Enter the necessary details
  • Click: Add STFAP Bracket button

    Modifyng STFAP Bracket

  • To modify an STFAP Bracket, Go to : Osa Staff functions
  • Click: STFAP Bracket Management Module
  • Go to Edit STFAP Bracket
  • Enter the STFAP Bracket number in the search box
  • Click: Search STFAP Bracket button
  • Click: Edit STFAP Bracket button
  • Enter all appropriate data
  • Click: Edit STFAP Bracket button

    Removing STFAP Bracket

  • To remove an STFAP Bracket, Go to : Osa Staff functions
  • Click: STFAP Bracket Management Module
  • Go to Remove STFAP Bracket
  • Enter the STFAP Bracket Number in the search box
  • Click: Remove STFAP Bracket button
  • Click the YES button to proceed

    View STFAP Bracket List

  • To view an STFAP Bracket, Go to : Osa Staff functions
  • Click: STFAP Bracket Management Module
  • Go to View STFAP Bracket List

    Modify Student's STFAP Bracket

  • To modify a student's STFAP Bracket, Go to : Osa Staff functions
  • Click: STFAP Bracketing
  • Go to Edit Student Bracket
  • Enter the Student ID and the new STFAP Bracket ID
  • Click: Edit Student Bracket button
