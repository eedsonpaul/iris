  • For Clerk:

      Adding a Degree Program:

    • To add a degree program, Go to: Degree Program Management Module -> Add Degree Program
    • Fill in all the required fields
    • Click "Add Degree Program" button

      Modifying a Degree Program:

    • To Modify a degree program, Go to: Degree Program Management Module -> Edit Degree Program
    • Input the degree program code on the textbox
    • Click "Edit" button
    • You will be directed to another page where you have to input all required fields
    • When done, Click "Modify Degree Program" button

      Removing a Degree Program:

    • To remove a degree program, Go to: Degree Program Management module -> Remove Degree Program
    • Input the dergree program code
    • Click "Remove Degree Program" button

      View Study Plan or Curricula per Degree Program:

    • To view study plan or curricula of a degree program, Go to: Degree Program Management module -> View Study Plan or Curricula

      Adding a course:

    • To add a course, Go to: Course module -> Add Course
    • Fill in all the required fields
    • Click "Add Course" button

      Modifying a course:

    • To Modify a course, Go to: Course module -> Edit Course
    • Input the course code on the textbox
    • Click "Edit" button
    • You will be directed to another page where you have to input all required fields
    • When done, Click "Modify Course" button

      Removing a course:

    • To remove a course, Go to: Course module -> Remove Course
    • Input the course code
    • Click "Remove Course" button
