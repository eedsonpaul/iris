SYSTEM RELEASE NOTES

UP Integrated Registration Information System (UPIRIS) Version 1.6

Date Released: October 11, 2010

Introduction

UP IRIS is an integrated online registration system. This is exclusive for the use of UP Cebu students, faculty, and staff. The UP IRIS allows students to register for enrollment through online access.  Official updates released by the university can be viewed through the system.  It also contains profiles and records of the people directly involved in the course registration process.  This System Release Notes will give you a brief overview of what the system functionalities are.

 

Server Minimum Requirements

Miminum:
  • Apache 2.2
  • PHP 5.0
  • MySQL 5.1
 
Recommended:
  • Apache 2.2.11
  • PHP 5.2.9
  • MySQL 5.1.36
 
               PHP 5.3 is highly discouraged since several functions in 5.2.9 are not present in PHP 5.3.               This version has only been tested using Mozilla Firefox 3.6.  It is highly recommended that this browser be used in working with the system.  This version has only been tested with Microsoft XP.  It is highly recommended that this operating system be used in working with the system.
 
Installation
               Installation is mainly done in the deployment server of UPIRIS.  Please refer to Administrator Manual for more information.
 
§         Install Wampserver 2.0i.  Installation notes are found in the following link: http://www.tinkertech.net/tutor/wamp/index.html
 
§         Found in the installation CD is the SQL file, db_softeng2010.  This file should be imported to phpMyAdmin.  This is the main database file.
 
§         Found in the installation CD as well is the Iris folder which contains all the files for the website.  Load this in the www directory under C:/wamp.
 
§         Set the Wampserver to Online to view the website under localhost/iris or http://202.90.133.58/iris/
